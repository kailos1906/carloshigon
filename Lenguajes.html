¿Que lenguajes manejo?

JAVA

Se constituye como un lenguaje orientado a objetos, su intención es permitir que los desarrolladores de aplicaciones escriban el programa una sola vez y lo ejecuten en cualquier dispositivo.

HTML

Es un lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet. Se trata de la siglas que corresponden a HyperText Markup Language

CSS

La hoja de estilo en cascada o CSS (siglas en inglés de cascading style sheets) es un lenguaje usado para definir la presentación de un documento estructurado

PYTHON

Es un lenguaje de programación versátil multiplataforma y multiparadigma que se destaca por su código legible y limpio.

C

lenguaje de programación general, ofrece un control absoluto sobre los programas que crea.
